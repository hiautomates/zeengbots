Preguntas Frecuentes

¿Necesito conocimientos técnicos para usar ZeengBots?
No. ZeengBots se configura solo con IA: le cuentas sobre tu negocio y la plataforma crea el agente, las respuestas y el flujo por ti. Todo sin escribir una línea de código.
¿Qué canales puedo conectar?
WhatsApp (QR y API oficial), Telegram, Instagram, Facebook Messenger y Web Chat para tu sitio. Todos los canales llegan a la misma bandeja unificada.
¿El bot puede tomar pedidos y agendar citas?
Sí. El agente puede mostrar tu catálogo, registrar pedidos completos y agendar citas con integración a Google Calendar (plan Pro), confirmando todo con el cliente de forma natural.
¿Puedo intervenir en las conversaciones?
Claro. Con el Chat en Vivo puedes pausar el bot en cualquier conversación, atender personalmente y devolver el control a la IA cuando quieras, sin perder el contexto.
